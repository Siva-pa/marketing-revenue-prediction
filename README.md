# 📊 Marketing Revenue Prediction System

## 🚀 Project Overview

This project predicts sales revenue using marketing, pricing, and customer data.

## 🧠 Model

* K-Nearest Neighbors (KNN) Regression
* Feature Scaling using StandardScaler

## 📊 Features

* EDA Dashboard (Univariate, Bivariate, Heatmap)
* Interactive Streamlit App
* Revenue Prediction System

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 💡 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
